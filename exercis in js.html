the Five ways to display JS Code

  1. console.log
    The console.log method is used to display messages or variables in the browser's console, which you can access through the developer tools (DevTools).
  2. Alert Box:
    The alert method displays a message in a modal dialog box, which must be dismissed by the user before they can interact with the webpage again.
  3. Document Write:
    The document.write method writes directly to the HTML document. However, it's not recommended for modern web development because it can overwrite the entire document if used after the page has loaded.
  4. DOM Manipulation:
    By selecting an element in the HTML document using methods like getElementById, you can change its content using the innerHTML property.
  5. Browser Console in HTML:
    You can also display JavaScript output in the browser's console by linking a function to an event, such as onload in the "body" tag.

    Each method is useful in different contexts, so it's great to know various ways to display or debug JavaScript code!

ten reserved JS keyword and their action

  1. break:
    The break statement is used to terminate a loop, switch, or label statement. When encountered, the control of the program moves out of the loop.
  2. case:
    The case keyword is used in switch statements to define various possible values that switch can match. If the value matches, the associated block of code is executed.
  3. catch:
    The catch block is used in conjunction with try to handle exceptions. If an error occurs in the try block, the catch block executes.
  4. class:
    The class keyword is used to define a class. Classes are templates for creating objects and can include methods and constructors.
  5. continue:
    The default keyword is used in switch statements to specify the code that should run if no case matches the switch expression.
  6. delete:
    The delete keyword is used to remove a property from an object.
  7. finally:
    The finally block is used in conjunction with try and catch to execute code regardless of whether an error occurred or not.
  8. function:
    The function keyword is used to declare a function.
  9. return:
    The return statement is used to exit a function and return a value.

These keywords are essential to understanding and writing JavaScript code effectively.

What are the differences and similarities between let, var and const?

Similarities

  • Variable Declaration:
    All three (let, var, const) are used to declare variables.
  • Storing Any Data Type:
    They can store any type of data, including strings, numbers, objects, arrays, etc.

Differences

  • let:
    Has block scope. It is only accessible within the block (e.g., { ... }) it is declared. let: Cannot be re-declared within the same scope.
  • var:
    Has function scope. It is accessible anywhere within the function it is declared. var and let: Can be assigned new values after their initial declaration.
  • const:
    Has block scope, similar to let. It is only accessible within the block it is declared. const:Cannot be re-declared within the same scope.
